# 焕甲

“焕甲”是一款具备美甲内容社区属性的一站式 AI 美甲服务产品。

用户可以在 App 中浏览、搜索各式美甲，通过 AI 小嘉（推荐系统）找到自己心仪的美甲款式，也可以分享自己的美甲作品。面对心仪的美甲，用户可以使用 AI 试戴功能，提前预览不同款式在自己手上的真实效果。试戴满意后，用户可以在线上预约门店，并到线下完成美甲服务。焕甲实现了从内容浏览、AI 试戴、线上预约到线下履约的完整服务闭环。

运营端面向平台运营人员，提供美甲内容社区管理、用户与商家数据查看等功能。同时，我们用 OpenClaw 为运营赋能。通过定制 Skill，定期追踪小红书上的美甲流行趋势，每周生成运营报告，辅助运营人员制定推款和运营策略，并将热门款式推送给商家，帮助商家更快了解市场偏好，提升服务承接和转化效率。

## 产品亮点

- 美甲内容社区：支持浏览、搜索各式美甲，也支持用户分享自己的美甲作品。
- AI 小嘉推荐：根据用户偏好和提问，推荐更符合需求的美甲款式。
- AI 试戴预览：上传手图后识别指甲区域，提前预览美甲上手效果。
- 预约履约闭环：用户选定款式后可线上预约门店，并在线下完成美甲服务。
- 运营趋势赋能：通过 OpenClaw 定制 Skill 追踪小红书美甲趋势，生成运营报告，并向商家推送热门款式。

## 美甲分割模型

我们训练了一个面向 AI 试戴场景的美甲分割器，用于识别用户手图中的指甲区域，并生成后续换甲所需的精细 mask。模型基于约 3 万份指甲图像与分割标注训练，能够在不同手型、拍摄角度和光照条件下稳定定位指甲区域，为试戴效果提供更准确的编辑范围。

模型 Demo：<https://huggingface.co/spaces/dongli/nail_yolo26>
